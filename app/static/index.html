Brand Events Finder

Поиск событий, влияющих на бизнес-метрики брендов

Как это работает: вы вводите названия брендов, указываете отрасль и выбираете типы событий. Сервис ищет в новостных источниках события, которые могли повлиять на бизнес-метрики: выручку, узнаваемость, репутацию, операционную деятельность.

AI-агент (Mistral) фильтрует результаты и оставляет только релевантные события. Для этого нужен бесплатный API-ключ — получите за минуту на console.mistral.ai. Без ключа результаты будут менее точными.
Для работы с Mistral API из России может потребоваться VPN.
